What is HTML?

HTML es el lenguaje que se usa para crear páginas web. Se encarga de la estructura y sostiene todo lo que después se ve en pantalla.

Explain the head and the body tags.

El head es el que contiene las instrucciones y el body donde si se muestran fisicamente los textos, imágenes, etc.

Explain the section, article, aside, header, footer tag?

Parte Explicación
Header es la parte de arriba que presenta el contenido
Footer es la parte de abajo que lo cierra
Section es un grupo de contenido relacionado
Article es un contenido que puede existir por sí solo
Aside es información extra que va al lado del contenido principal.

What is CSS?

CSS es el lenguaje que le da el estilo a una página web.

What is the purpose of CSS?

Sirve para:

  • Agregar colores
  • Cambiar fuentes
  • Ajustar tamaños

How do you apply a CSS to an HTML (explain the 3 options)

  1. Escribir el estilo directamente en la etiqueta.
  2. Escribir el CSS dentro del Head.
  3. Crear un archivo .css y conectarlo al HTML.

What is a selector? What is a CSS rule?

Es la parte que le dice a CSS qué elemento va a cambiar y la regla incuye al selector junto con los estilos que se van a aplicar.

Explain a: Tag Selector, Class Selector, Tag and Class Selector, Id Selector

  • Tag Selector es cuando aplicas estilos a una etiqueta directamente usando su nombre.
  • Class Selector se usa para aplicar estilos solo a los elementos de una clase especifica, se usa un punto antes del nombre.
  • Tag and Class Selector es cuando combinas una etiqueta y una clase para aplicar estilos solo a esa etiqueta que tenga esa clase.
  • Id Selector se usa para aplicar estilos solo a un elemento usando su ID, se usa # antes del nombre, un ID no se debe repetir en la misma pagina.

What is a class? What is an id? How are they related to CSS?

Una clase es un atributo de HTML que nos sirve para identificar varios elementos y aplicarles el mismo estilo.
Un ID es un atributo de HTML que identifica un elemento único dentro de la página, solo debe de existir uno con ese nombre.
Ambos se relacionan a CSS ya que se usa selectores para elegir que elementos se van a estilizar, clase se usa un punto y ID con #, CSS busca en el HTML los elementos con esa clase o ese ID y aplica los estilos.

Explain the CSS Box Model

Es una forma en la que CSS organiza y calcula el espacio que ocupa cada elemento en una página. Tiene 4 partes: Content que es lo que esta adentro del elemento, Padding que es el espacio entre el contenido y el borde, Border que es la línea al rededor del elemento y Margin que es el espacio fuera del elemento.

Explain the media queries

Se usa para aplicar diferentes estilos dependiendo del dispositivo o el tamaño de la pantalla, ayuda a hacer las páginas sensibles. Se usa un @media (condición).

Explain display: grid

Es una propiedad de CSS que permite organizar elementos en filas y columnas, sirve para crear layouts ordenados y responsivos.

Explain display: flex

Es una propiedad de CSS que organiza y alinea elementos dentro de un contenedor de forma flexible. Se usa para centrar cosas, hacer menús o acomodar elementos fácilmente.
